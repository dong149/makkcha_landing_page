## Description
### 🚌️ 귀가의 모든 것 "막차"

2021.5.24 기준 누적 사용자 약 5 만명

대중교통 막차정보를 카운트다운 형태로 제공하고,<br/>
막차 시간에 집에 귀가할 수 있는 다양한 옵션을 제공하는 서비스.
현재 위치를 기준으로 설정한 목적지까지의 모든 수단을 알려드립니다!

## Feature
- 버스,지하철,버스+지하철 등 모든 대중교통 막차 시간 확인
- 심야버스, 택시 정보 확인
- 히스토리 기능을 지원하여 최근 검색 기록 중 선택
- 통금시간 설정 기능
- 근처 따릉이 정보 제공
